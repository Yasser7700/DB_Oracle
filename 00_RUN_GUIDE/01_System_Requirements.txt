📌 نظام التشغيل:
   • Windows 10/11 Professional أو Enterprise (64-bit)
   • أو Windows Server 2016/2019/2022
   • أو Oracle Linux 7/8
   • أو Red Hat Enterprise Linux 7/8

📌 المعالج (CPU):
   • الحد الأدنى: Intel Core i5 أو AMD Ryzen 5
   • الموصى به: Intel Core i7/i9 أو AMD Ryzen 7/9
   • عدد الأنوية: 4 كحد أدنى، 8 موصى به

📌 الذاكرة (RAM):
   • الحد الأدنى: 8 GB
   • الموصى به: 16 GB أو أكثر
   • للإنتاج: 32 GB أو أكثر

📌 مساحة التخزين:
   • الحد الأدنى: 50 GB مساحة حرة
   • الموصى به: 100 GB SSD
   • للإنتاج: 500 GB SSD/NVMe

✅ Oracle Database 19c Enterprise Edition
   • الإصدار: 19.3.0.0 أو أحدث
   • تأكد من تفعيل الخيارات التالية:
     - Oracle Label Security
     - Oracle Database Vault (اختياري)
     - Oracle Advanced Security

✅ Oracle SQL Developer
   • الإصدار: 21.x أو أحدث
   • تحميل من: https://www.oracle.com/tools/downloads/sqldev-downloads.html

✅ SQL*Plus
   • يأتي مع تثبيت Oracle Database
   • تأكد من إضافته لمتغير PATH

✅ Java Development Kit (JDK)
   • الإصدار: 8 أو 11 (مطلوب لـ SQL Developer)

📋 متغيرات البيئة المطلوبة:
   ORACLE_HOME     = C:\app\oracle\product\19.0.0\dbhome_1
   ORACLE_SID      = ORCL (أو اسم قاعدة البيانات الخاصة بك)
   PATH            = %ORACLE_HOME%\bin;%PATH%
   NLS_LANG        = AMERICAN_AMERICA.AL32UTF8

📋 خدمات Windows المطلوبة:
   • OracleServiceORCL (أو اسم خدمتك)
   • OracleOraDB19Home1TNSListener

📋 منافذ الشبكة:
   • منفذ Listener الافتراضي: 1521
   • منفذ EM Express: 5500 (اختياري)

🔍 خطوات التحقق:

1. تحقق من تشغيل قاعدة البيانات:
   افتح CMD كمسؤول واكتب:
   > sqlplus / as sysdba
   SQL> SELECT STATUS FROM V$INSTANCE;
   
   النتيجة المتوقعة: OPEN

2. تحقق من إصدار قاعدة البيانات:
   SQL> SELECT BANNER FROM V$VERSION;
   
   النتيجة المتوقعة: Oracle Database 19c Enterprise Edition

3. تحقق من مساحة التخزين:
   SQL> SELECT TABLESPACE_NAME, 
              ROUND(SUM(BYTES)/1024/1024,2) AS SIZE_MB
        FROM DBA_DATA_FILES
        GROUP BY TABLESPACE_NAME;

4. تحقق من المستخدمين الأساسيين:
   SQL> SELECT USERNAME FROM DBA_USERS 
        WHERE USERNAME IN ('SYS','SYSTEM');

⚠️ تحذيرات:
   • لا تستخدم كلمات مرور افتراضية في بيئة الإنتاج
   • تأكد من وجود نسخة احتياطية قبل البدء
   • احفظ جميع كلمات المرور في مكان آمن
   • لا تشارك بيانات الاتصال مع أشخاص غير مصرحين

💡 نصائح:
   • استخدم SSD للحصول على أفضل أداء
   • خصص ذاكرة كافية لـ SGA و PGA
   • راقب مساحة التخزين بانتظام
   • احتفظ بسجل لجميع التغييرات
